# DeliverryApp
est un site web créé pour les personnes qui ont besoin d'une livraison rapide, sans tracas.
Il suffit de créer votre commande et d'attendre qu'un livreur vous propose un prix.
Vous bénéficiez du meilleur prix grâce à la concurrence entre tous les livreurs.
